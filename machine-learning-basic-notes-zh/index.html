频率派和贝叶斯派

频率派

频率派是把参数\(\theta\)认为是一个未知的定值来对\(p(Y|X; \theta)\)建模，求最大似然估计(MLE)。在已知\(Y\)和\(X\)的情况下，默认认为噪声\(\epsilon\)符合正态分布，并使\(\epsilon\)最小，求出\(\theta\)的最大似然估计，便是线性回归中的最小二乘法。当然实际中还会加入regularization，包括常见L1和L2两种方式。

神经网络属于频率派的方式。

贝叶斯派

贝叶斯派则是把参数\(\theta\)也认为是一个参数，因此目标就是对\(p(\theta|Y,X)\)建模，求出\(\theta\)的分布，使用贝叶斯公式可以进一步得到： \[p(\theta|Y,X) = \frac{p(\theta,Y|X)}{p(Y|X)} = \frac{p(Y|\theta,X)p(\theta|X)}{p(Y|X)}\] 上面的式子我们不需要全部计算，可以认为等于下式子： \[p(\theta|Y,X) = p(Y|\theta,X)p(\theta)\] 其中\(p(\theta)\)也被叫做先验。

更多时候，还是需要让\(\theta\)成为一个定值来使用，因此在得到了\(\theta\)的分布之后，需要使用最大后验概率估计(MAP)，选取\(\theta\)概率最大的那个点(比如正态分布的均值点)，来作为\(\theta\)的值进行使用。
